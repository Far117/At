◆ AtScreen

typedef struct AtScreen AtScreen

The backbone of the @ library.

Allows information to be drawn to the console based on x/y coordinates, rather than line-by-line.

Warning
By necessity, an AtScreen is internally referenced as [y][x] instead of [x][y]. Since the latter is more natural, all functions that require x/y coordinates will switch their inputs (x->y, y->x) to account for this quirk. When creating a user-defined function, care must be taken to reference the AtScreen in a similar way.
